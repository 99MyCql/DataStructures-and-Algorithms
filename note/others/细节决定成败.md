# 细节决定成败

## 边界判断在前，逻辑判断在后

错误做法，会触发越界：

```c
while (nums[j] == nums[p] && j < len) j++;
```

正确做法：

```c
while (j < len && nums[j] == nums[p]) j++;
```

## C++ long long 最大值

0x7fffffffffffffff 或 LLONG_MAX

## C++ 运算符优先级

乘除/取余 > 加减 > 左移/右移 > 大于/小于/等于逻辑比较 > 位运算 > 与或逻辑运算

示例：

```C++
(a & b) != 0; // 此处位运算必须加括号
```
